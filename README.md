Dredd::Rack Rails Example
=========================

Demonstrate the [Dredd::Rack][dredd-rack] usage in a Rails app.

  [dredd-rack]: https://github.com/gonzalo-bulnes/dredd-rack

Usage
-----

```bash
# run the app
rackup -p 3000
```

Development
-----------

```bash
# run the app
rerun 'rackup --port=3000'
```
